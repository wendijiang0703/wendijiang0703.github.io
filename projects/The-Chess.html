The Chess

Three-Dimensional Robotic Interaction

Design Based on Computer Vision

#ROS #robotic

2024.4~2024.10
Individual Project

This project investigates the dynamic interaction between human creativity and machine rationality in a co-design process. Using a board game design facilitated by a robotic arm, the experiment explores how real-time feedback from the machine can influence and shape human decision-making, and vice versa. A three-dimensional map was created on stacked blocks that were assigned three different colors for the robot to tell the graphic location of each step. By treating the robotic arm as an active participant rather than a passive tool, the project seeks to understand how attitudes and strategies evolve during interaction. The research aspires to uncover how this bi-directional feedback loop can foster new paradigms in collaborative design, blending the machine’s precision with the human’s emotional sensitivity.
